# Datenschutz & Transparenz

Dieses Tool wurde speziell entwickelt, um Nutzern eine vertrauenswürdige Option zu bieten.

## Garantien

- **Es werden keine Daten versendet** — alles läuft zu 100 % in deinem Browser.
- **Keine Telemetrie** — keine Analytics, keine Tracking-Skripte.
- **Nur GitHub Pages** — der Code, den du im Repository siehst, ist genau das, was ausgeführt wird.
- Tab schließen oder Seite neu laden → alle Daten sind weg.

## Warum eine separate statische Version?

Gehostete Versionen auf eigenen Domains (auch die Originalversion) könnten theoretisch Tracking oder Backend-Aufrufe enthalten. Durch die Veröffentlichung eines rein statischen Builds auf GitHub Pages machen wir es strukturell unmöglich, Daten nach außen zu senden.
