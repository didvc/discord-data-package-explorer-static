# Erste Schritte

## 1. Dein Discord-Datenpaket anfordern

1. Öffne Discord (Desktop oder Web).
2. Gehe zu **Benutzereinstellungen → Datenschutz & Sicherheit**.
3. Klicke auf **Daten anfordern**.
4. Stelle sicher, dass **Nachrichten** und **Kontoinformationen** ausgewählt sind.
5. Warte auf die E-Mail (kann mehrere Stunden bis Tage dauern).

## 2. Herunterladen und verwenden

1. Lade die Datei `package.zip` aus der E-Mail herunter.
2. Öffne die [Live-App](https://didvc.github.io/discord-data-package-explorer-static/).
3. Ziehe die Zip-Datei per Drag & Drop hinein (oder klicke zum Auswählen).

Fertig — alles läuft in deinem Browser.

## Hinweise

- Das Tool benötigt nur die Ordner `messages/` und die Kontoinformationen.
- Große Pakete können etwas länger zum Parsen brauchen.
- Es werden keine Daten an Server gesendet.
