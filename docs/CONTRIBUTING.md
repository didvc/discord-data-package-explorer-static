# Contributing to the Documentation

The documentation is hosted at https://discord-dpes.pages.dev/

## Languages

We currently maintain:
- English (primary)
- German

Placeholders exist for French, Russian, and Chinese.

Additional language contributions are welcome.

## How to contribute

1. Edit the Markdown files under `docs/`
2. For new languages, add the locale in `.vitepress/config.mts`
3. Keep technical terms (DM, package.zip, etc.) consistent where possible
4. Open a Pull Request
