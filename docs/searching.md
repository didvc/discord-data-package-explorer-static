# Searching Your Messages

The main feature of this tool is full-text search across **all** your messages.

## How to search

- Type any keyword in the search box.
- Use the filters to narrow results:
  - **All channels**
  - **Servers only**
  - **DMs only**
- Filter by channel name
- Sort by newest or oldest

## Export

You can export the current filtered results as CSV using the **Export CSV** button.

## Tips

- The search works on the full message content.
- Results are limited in the UI for performance (the underlying data is all there).
- Close the tab to completely clear the data from memory.
