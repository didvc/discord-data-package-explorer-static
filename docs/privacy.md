# Privacy & Transparency

This tool was created specifically to give users a high-trust option.

## Guarantees

- **No data is sent anywhere** — everything runs 100% in your browser.
- **No telemetry** — no analytics, no tracking scripts.
- **GitHub Pages only** — the code you see in the repository is exactly what runs.
- Close the tab or refresh the page → all data is gone.

## Why a separate static version?

Hosted versions on custom domains (even the original) can theoretically include tracking or backend calls. By publishing a pure static build on GitHub Pages, we make it structurally impossible to phone home.
