# Documentation for Discord Data Package Explorer (Static)

This folder contains the source for the documentation site.

**Live docs:** https://discord-dpes.pages.dev/

## Building locally

```bash
npm run docs:dev
```

## Building for production

```bash
npm run docs:build
```

The output goes to `docs/.vitepress/dist/`.

You can deploy this folder to Cloudflare Pages, Netlify, Vercel, or any static host.

## Languages

- English (default)
- German (partially translated)
- French, Russian, Chinese (structure prepared)
